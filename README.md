# Jupyter Exploration

## Overview
This repository was created as part of a lab assignment to explore GitHub and Jupyter Notebooks. The purpose of this project is to demonstrate basic version control skills using GitHub and introductory usage of Jupyter Notebooks for interactive computing with Python.

## Repository Contents
- **README.md**  
  Provides an overview of the project and its purpose.
  
- **My_First_Notebook.ipynb**  
  A Jupyter Notebook containing:
  - A Markdown cell with descriptive text
  - A Python code cell that outputs text

## Tools Used
- GitHub (repository hosting and version control)
- VS Code (for cloning repos and eidting/uploading files)
- Jupyter Notebook (via local installation)
- Python (basic output example)

## What I Learned
Through this lab, I learned how to:
- Create and manage a GitHub repository
- Make commits and push changes to GitHub
- Create and run cells in a Jupyter Notebook
- Use Markdown and Code cells in Jupyter
- Upload notebook files to GitHub using VS Code
